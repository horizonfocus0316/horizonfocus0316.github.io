了解記者使命
About

新聞記者在做什麼？

記者的工作，不只是「把事件寫出來」，而是釐清利害關係人、查證與交叉比對、把複雜現象翻成可理解的公共知識。 在速度與正確之間取得平衡，是每天都要面對的專業選擇。

  • 確立公共意義的選題與角度
  • 多源查證與資料佐證（數據、文件、實地）
  • 以倫理為底線的拍攝、採訪與呈現
Workflow

採訪到成稿：6 步驟

1. 選題與假設

定義公共關切、列出初步假設與可能的利害人圖譜。

2. 資料蒐集

法規、統計、開放資料；同時建立時序表與關鍵事件清單。

3. 田野採訪

現場走訪、錄音錄影、比對受訪者觀看與非語言線索。

4. 多源查證

至少雙方說法 + 文件佐證；對爭議點逐條回覆與補強。

5. 結構與寫作

倒金字塔 / 敘事線；小標、圖表與引述的節奏配置。

6. 編輯與發布

事實核查、法務風險檢視、標題與社群貼文的資訊準確。
