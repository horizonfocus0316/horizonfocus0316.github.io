Bootstrap sticky-top × 控制閱讀時間

內層 sticky-top 負責「黏住」，外層容器的高度（例如 300vh）負責「拖長觀看時間」。 往下捲吧！

開始

章節一：圖像黏頂示範

外層 300vh → 你要捲 3 倍的距離，這張圖才會結束。

章節二

把敘事分段，每段各有一張圖／一段影片。

章節三

這種手法很適合做時間線、旅程、或專題報導的章節。

章節四

結尾段：提示讀者下一步行動或補充延伸閱讀。

雙欄黏頂（文字 × 影像）

這裡放你的敘事文字。當右邊的影片/圖片黏在頂端時，讀者可以邊讀邊看。

如果想做「左右交替」的效果，就做多段區塊：一段右黏頂、一段左黏頂。

每段高度不夠？用 padding 或者把文章切成多段各自成為一個 section。

想更精緻，可以讓圖片換場時做淡入淡出（純 CSS 也可）。

這段只是示意，實作時換成你的專題文字。

右邊這塊會黏住，左邊文字捲動。
